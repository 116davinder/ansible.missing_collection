.. contents::

Synopsis
--------
- Get information from Consul (Health).
- https://www.consul.io/api-docs/health



Requirements
------------
The below requirements are needed on the host that executes this module.

- requests

Examples
--------

.. code-block:: yaml

    - name: get health of given node
      consul_health:
        get_node_health: true
        node: "server-1"
        dc: "dc1"
